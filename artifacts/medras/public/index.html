AI assistant
Which test should I use for two unpaired non-normal groups with n=28?
Use the Mann-Whitney U test (Wilcoxon rank-sum). With n=28 per group and non-normal distribution confirmed by Shapiro-Wilk (p<0.05), a parametric t-test would be inappropriate. MedRAS Sigma will run this automatically.
How do I report the effect size for this?
Report r = Z / √N (rank-biserial correlation). Values: small ≥0.1, medium ≥0.3, large ≥0.5. Sigma includes this in the output table automatically.

About MedRAS

Not a chatbot. A research operating system.

MedRAS is purpose-built for medical, biomedical, and academic researchers who need methodological rigour at every step. The interface is structured and menu-driven; statistical computation is performed by validated libraries, not by language models. AI assists with planning and writing — never with numerical results.
