      What type of genetic study?
      GWAS uses genome-wide significance threshold: α = 5×10⁻⁸ (not 0.05). Typical sample sizes: 5,000 to 100,000+. This is biobank-level recruitment.
      Linkage analysis uses LOD score ≥ 3.0 as genome-wide significance threshold, not a p-value. Sample size depends on family structure and disease model. Use SIMLINK or GENEHUNTER software for precise calculation.
      Complex-trial design adjustments

      If unsure, use 0.40 as a conservative estimate.

      Haematological markers: 0.60. Clinical outcomes: 0.50. Repeated measures REDUCE required n — this is a statistical efficiency gain.

      Lab/blood: 0.05. Clinical: 0.10. Behavioural: 0.15. Multicentre design INFLATES required n due to between-site variation.

      Adaptive penalty preserves overall α across interim looks.
