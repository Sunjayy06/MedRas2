Build your own practice dataset

Answer 5 quick questions. We'll generate fake patient data with the variables you describe — nothing is uploaded.

1 2 3 4 5

1. What variables do you want?

Type one variable name per line. Examples: age, sex, BMI, treatment_group, Harris Hip Score. We'll auto-detect what kind each one is in the next steps.

2. What's the sample size?

How many patients should we generate? Pick a number between 20 and 500. Around 60 is a comfortable default for practice.

3. Any specific ranges or values per variable? (optional)

Set min/max for numeric variables, percentages for binary, or category labels for nominal. Leave anything blank to use sensible defaults. You'll classify each variable's full type later in Step 3 — Variables.

4. What result are you expecting? (optional)

Tell us in a sentence which variable should differ between groups and by roughly how much. We'll bake this signal into the data so your analysis can find it. Leave blank for random/no-effect data.

5. Any specific instructions for the Excel sheet? (optional)

Free-form notes — write anything you'd like the data to reflect. We'll do our best to honour it. Examples: "male population should be higher than female", "include some missing values in BMI", "results should favour the treatment group", "keep ages between 50 and 80".

Step 5 · Check normality

For each scale variable we run Shapiro-Wilk (n < 50) or Kolmogorov-Smirnov (50–2000). Above 2000 we use shape thumb-rules instead. Override the verdict if you have a strong reason.
