**On multiple outcomes — honest answer:**

Currently the system is built for one outcome variable. The correlation study runs pairwise analysis of one outcome against every other variable. Multiple outcomes are not supported yet. A researcher studying both ER expression and tumour recurrence as two separate outcomes would need to run the analysis twice.

Whether to add it now or later depends on how common it is for your users. For most standard medical studies one primary outcome is the norm. Secondary outcomes are usually analysed separately. I would not add it now — add it after launch when a real user requests it.
