Create a function called detect_domain that takes the user role, selected format and research topic as inputs and returns the correct domain string. Use simple keyword matching on the research topic to confirm the domain. For example if the format is ICMR or the topic contains words like patients, clinical, disease, treatment, hospital, drug then confirm medical_clinical domain. If topic contains words like algorithm, software, machine learning, circuit then confirm computer_science. If topic contains words like market, revenue, firm, company, profit then confirm business_economics. If no keywords match use the format selected to determine domain. Return the matched domain string.

Create a function called get_databases_for_domain that takes a domain string and returns the list of databases to search from DOMAIN_DATABASE_MAP.
