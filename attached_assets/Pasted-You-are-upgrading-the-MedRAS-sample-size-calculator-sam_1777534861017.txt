════════════════════════════════════════════════════════

When a research objective is pasted, automatically 
scan for these keywords and set flags:

Groups detection:
  "two groups" / "two arms" / "compare" / "versus" /

  "k groups" → MULTI_GROUPS = true

  "mean" / "average" / "score" / "level" / "mg/dL" /

  "correlation" / "association" / "relationship" → CORRELATION = true
  "survival" / "time to event" / "hazard" → SURVIVAL = true

  "baseline" + any week/month/timepoint mention → LONGITUDINAL = true
  "follow-up" / "longitudinal" / "trajectory" / 
  "timepoints" / "weeks" / "months" → LONGITUDINAL = true

Complexity detection:
  "multicentre" / "multicenter" / "multiple sites" /

  "SNP" / "gene" / "genetic" / "allele" / "GWAS" /

  "elderly" / "≥65" / "≥ 65" / "paediatric" / 
  "children" → HIGH_ATTRITION = true
  "qualitative" / "focus group" / "interview" → QUALITATIVE = true
  "pilot" / "feasibility" → PILOT = true
  "in vitro" / "in-vitro" / "cell line" / 
  "culture" → INVITRO = true
  "in vivo" / "animal" / "murine" / "rat model" → INVIVO = true

  QUALITATIVE → n = 12–15, show saturation note
  PILOT → n = 25 total, show pilot note

  SURVIVAL → Log-rank / Survival analysis
  CORRELATION → Pearson/Spearman correlation formula
  MULTI_GROUPS + MEAN → One-way ANOVA
  TWO_GROUPS + PROPORTION → Two independent proportions
  TWO_GROUPS + MEAN → Two independent means
  ONE_GROUP + PROPORTION → One proportion
  ONE_GROUP + MEAN → One sample mean
  Default fallback → Two independent proportions

After parsing, show a small "Detected:" chip row

  [2 groups] [Proportions] [Longitudinal] [Multicentre]
  "Based on your objective, we selected: 
   Repeated Measures — Two Groups"
  [Change formula ▾] (dropdown to override)

CHANGE 4 — COMPLEX TRIAL LAYER PIPELINE
════════════════════════════════════════════════════════

Activate ONLY when researcher selects

Do NOT activate for Simple or Standard modes.

Show these additional fields in sequence, 
revealing each only if the previous answer

FIELD SET A — always show in complex mode:
  - Number of primary outcomes 
    (dropdown: 1, 2, 3, 4, 5+)
  - Number of timepoints 
    (number input, min 1)
  - Is this multicentre? (Yes / No toggle)
  - Is this an adaptive design? (Yes / No toggle)

FIELD SET B — show if primary outcomes ≥ 2:
  - "Estimated correlation between your 
     primary outcomes (0 to 1.0)"
    Default: 0.40
    Help text: "If unsure, use 0.40 as a 
    conservative estimate"
  Layer 2 activates: 
    α_adj = α / k_outcomes
    n_composite = n_base / ρ_min

FIELD SET C — show if timepoints > 1:
  - "Within-subject correlation across 
     timepoints (ρ)"
    Default: 0.60
    Help text: "For haematological markers 
    use 0.60. For clinical outcomes use 0.50"
  Layer 3 activates (reduces n):
    n = n / [1 + (m−1) × ρ_time]
  Show label: "Repeated measures reduces 
  required n — this is a statistical 
  efficiency gain."

FIELD SET D — show if multicentre = Yes:
  - Number of study sites (default 10)
  - ICC value (default 0.05)
    Help text: "Lab/blood outcomes: 0.05. 
    Clinical outcomes: 0.10. 
    Behavioural outcomes: 0.15"
  Layer 4 activates:
    DEFF = 1 + (n/sites − 1) × ICC
    n_multicentre = n × DEFF
  Show note: "Multicentre design INFLATES 
  required n due to between-site variation"

FIELD SET E — show if adaptive = Yes:
  - Number of planned interim analyses 
    (1, 2, or 3)
  Layer 6 activates:
    1 interim → multiply final n by 1.05
    2 interims → multiply final n by 1.07
    3 interims → multiply final n by 1.10
  Show note: "Adaptive penalty preserves 
  overall alpha across interim looks"

CHANGE 7 — MISSING FORMULAS IN DROPDOWN
════════════════════════════════════════════════════════

Add these formulas to the existing dropdown 
(they are missing from your current list):

1. Pearson/Spearman Correlation
   Formula: n = (Z_α/2 + Z_β)² / (0.5 × ln[(1+r)/(1−r)])² + 3
   Inputs: Expected correlation (r), alpha, power
   Dropout: HIDE (single timepoint)
   Ideal for: "Measuring strength of linear 
   relationship between two continuous variables"

2. Repeated Measures ANOVA
   Formula: n per group = (Z_α/2 + Z_β)² × 2σ² / 
   (δ² × m × (1−ρ))
   Inputs: number of timepoints (m), 
   within-subject correlation (ρ), 
   effect size (δ), SD (σ)
   Dropout: SHOW
   Ideal for: "Comparing groups measured 
   at 3+ timepoints — more powerful than 
   repeated t-tests"

3. Survival Analysis / Log-rank test
   Formula: d = (Z_α/2 + Z_β)² / 
   (log(HR))² × 4
   n = d / (p_event1 + p_event2)
   Inputs: Hazard ratio (HR), 
   event probability in each group,
   median survival times, 
   study duration, accrual period
   Dropout: SHOW
   Ideal for: "Time-to-event outcomes: 
   mortality, disease progression, 
   hospital discharge"

4. Bayesian Sample Size
   Formula: n based on posterior 
   credible interval width or 
   Bayes Factor threshold
   Inputs: Prior distribution parameters,
   desired credible interval width,
   minimum Bayes Factor (default 3)
   Dropout: SHOW
   Show note: "Bayesian estimates depend 
   heavily on prior specification. 
   Report prior assumptions clearly."
   Ideal for: "Studies with strong prior 
   evidence or adaptive Bayesian trials"

5. Non-inferiority Trial
   Formula: n = (Z_α + Z_β)² × 2σ² / 
   (δ − NI_margin)²
   Inputs: Non-inferiority margin (δ),
   expected difference, SD
   Note: Use one-tailed alpha (0.025 
   for 5% overall)
   Dropout: SHOW
   Ideal for: "Showing a new treatment 
   is not meaningfully worse than standard care"

6. Equivalence Trial
   Formula: n = (Z_α/2 + Z_β)² × 2σ² / 
   (EQ_margin − |δ|)²
   Inputs: Equivalence margin,
   expected difference, SD
   Dropout: SHOW
   Ideal for: "Showing two treatments 
   produce the same result (e.g. 
   generic vs branded drug)"

7. Diagnostic Accuracy Study
   Formula: n = Z² × Se × (1−Se) / d²
   (separate for sensitivity and specificity)
   Inputs: Expected sensitivity,
   expected specificity, 
   margin of error (d), 
   disease prevalence
   Dropout: HIDE
   Ideal for: "Validating a new diagnostic 
   test against a gold standard"

8. Intraclass Correlation Coefficient (ICC)
   Formula: n = 1 + (Z_α/2 + Z_β)² × 
   2(1−ρ₀)² / (ρ₁−ρ₀)²
   Inputs: Null ICC (ρ₀), 
   Alternative ICC (ρ₁), 
   number of raters (k)
   Dropout: HIDE
   Ideal for: "Measuring agreement between 
   raters or repeated measurements of 
   same instrument"
