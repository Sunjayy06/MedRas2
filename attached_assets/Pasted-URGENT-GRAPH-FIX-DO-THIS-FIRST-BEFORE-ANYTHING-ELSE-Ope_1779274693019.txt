When the result is a Chi-square test 
or Fisher exact test between two 
categorical variables, the graph 
must be a stacked bar chart showing 
the percentage of positive and 
negative outcome in each category 
side by side.

When the result is a Mann-Whitney 
or t-test comparing a continuous 
variable between two groups, the 
graph must be a box plot showing 
the distribution in each group with 
individual data points overlaid.

When showing the distribution of a 
single categorical variable like 
gender or tumour type, the graph 
must be a pie chart if there are 
two or three categories or a 
horizontal bar chart if there are 
four or more categories.

When showing the distribution of a 
single continuous variable like age 
or tumour size, the graph must be 
a histogram with a normal curve 
drawn over it.

When showing correlation between 
two continuous variables, the graph 
must be a scatter plot with a 
regression line.

The only situations where a forest 
plot is correct are logistic 
regression showing odds ratios and 
Cox regression showing hazard ratios. 
In all other situations use the 
graph types listed above.
